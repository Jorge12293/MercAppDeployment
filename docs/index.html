Jorge Rivera · UPS Cuarto Ciclo · Aplicaciones Web

Descripción del sistema

MercApp es una SPA (Single Page Application) de comercio electrónico que permite listar, filtrar, crear y editar productos por categoría, gestionar un carrito de compras persistente en sesión y visualizar historial de productos vistos recientemente.

Stack tecnológico

  • Frontend: Vue 3, Vue Router 5, Vite 5, Fetch API
  • Backend: Node.js, Express 5, Mongoose, Socket.io, Multer
  • Base de datos: MongoDB Atlas (clúster M0, región US‑East)
  • Despliegue: Railway (API), Netlify (SPA), GitHub Pages (micrositio)

Ejecución local

1. Requisitos

  • Node.js 18+
  • pnpm (npm i -g pnpm)
  • Cuenta MongoDB Atlas o instancia local

2. Backend

cd backend
cp .env.example .env        # Completar MONGODB_URI y SESSION_SECRET
npm install
npm run dev                 # http://localhost:3000

3. Seed (datos de prueba)

npm run seed

4. Frontend

cd frontend
cp .env.example .env.development   # VITE_API_URL=http://localhost:3000/api
pnpm install
pnpm run dev               # http://localhost:5173

5. Build de producción (frontend)

pnpm run build     # genera frontend/dist/
pnpm run preview   # previsualización local del build
